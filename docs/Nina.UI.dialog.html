Class: dialog

Nina.UI. dialog

new dialog(args)

Creates a DHTML based dialog. The dialog contains control boxes, a title bar, an entry in the Nina.UI.TaskBar if it's visible, resizable, control boxes etc. dozens of events to attach to.
Parameters:
Name Type Argument Description
args Native.Object <optional>
 Parameters for the dialog.
Properties
Name Type Argument Default Description
state Native.Number <optional>
 0 The state of the Nina.UI.dialog. 0: Normal, 1: minimized, 2: maximized, 3: hidden.
rect Nina.UI.Rect <optional>
 The Nina.UI.Rect of the Nina.UI.dialog.
resizable Native.Boolean <optional>
 true When set true the Nina.UI.dialog will be resizable. When set false the Nina.UI.dialog will be resizable.
showContentWhileMoving Native.Boolean <optional>
 false When set true content will be shown while moving and resizing. When set false a preview will be shown while moving and resizing.
centerVertically Native.Boolean <optional>
 false When true this Nina.UI.dialog will be centered vertically when the position is next updated or moveToCenter is called.
centerHorizontally Native.Boolean <optional>
 false When true this Nina.UI.dialog will be centered horizontally when the position is next updated or moveToCenter is called.
style Nina.UI.Style <optional>
 Nina.UI.Style The Nina.UI.Style of this Nina.UI.dialog.
modal Native.Boolean <optional>
 false When set true the Nina.UI.dialog will be in modal mode, appearing on top of everything and preventing access to background elements.
modalCanClose Native.Boolean <optional>
 false When set true, the close control box will be enabled even when in modal mode.
moveable Native.Boolean <optional>
 true When set false the Nina.UI.dialog cannot be moved.
alwaysOnTop Native.Boolean <optional>
 false When set true the dialog will appear above all other Nina.UI.dialog instances.
dontInit Native.Boolean <optional>
 false When set true the dialog will not initialize until you call the Nina.UI.dialog.Init method.
hidden Native.Boolean <optional>
 false When set true the dialog will be hidden.
hiddenFromTaskBar Native.Boolean <optional>
 false When set true the dialog will be hidden from the task bar.
title Native.String <optional>
 The title of the Nina.UI.dialog.

Examples
 ///Create a simple reference to a new dialog, set the title and make it modal.///
var myDialog = Nina.UI.dialog({
	title:'My Dialog',
	modal: true
});

Attach to an event when you create the dialog.

var myDialog = Nina.UI.dialog({
	title:'My Dialog',
	close:function(e,dialog){
	    dialog.title('Can\'t close me.');
		dialog.preventDefault();
		return;
	}
});

Attach to an event after you create the dialog

var myDialog = Nina.UI.dialog({
	title:'My Dialog'
});
myDialog.addEventListener('close',function(e,dialog){
  dialog.title('Can\'t close me.');
	dialog.preventDefault();
	return;
},false);
