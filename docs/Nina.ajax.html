Class: ajax

Nina. ajax

new ajax(args, options)

Sends structured JSON requests to a HTTP server running Nina.Core.

Parameters:
Name Type Argument Default Description

Request object or an array of request objects.

Options and events associated with the request collection.

The method to execute on the server. Should be in the format <namespace>.<class>.<method>. E.g.: Nina.Authentication.Logon. When in the Nina namespace the namespace can be omitted.

Examples

Create a request for a single method.

var foo = Nina.ajax({
   method: 'Authentication.CreateAccount',
   parameters: ['Test', '1234'],
   procedure: function (e) {
       alert(e.Message);
   }
});

Create a request for two methods.

var foo = Nina.ajax([
   {
       method: 'Authentication.CreateAccount',
       parameters: ['Test', '1234'],
       procedure: function (e) {
           alert(e.Message);
       }
   },
   {
       method: 'Authentication.Logon',
       parameters: ['Test', '1234'],
       procedure: function (e) {
           alert(e.Message);
       }
   }
]);

Create a request that contains a file and show the progress of the upload.

var foo = Nina.ajax({
   method: 'FileManager.Upload',
   parameters: ['~\\', document.getElementById('file1')],
   procedure: function (e) {
       document.body.innerHTML = e.Message;
   }
},
{
   progress:function (e) {
       document.body.innerHTML = Math.round((e.loaded / e.total) * 100) + ' %';
   }
});

Create a request that contains more than one file.

var x = Nina.ajax({
   method: 'FileManager.UploadFiles',
   parameters: [['~\\','~\\'], [document.getElementById('file1'),document.getElementById('file2')]],
   procedure: function (e) {
       showMessage(b, e.Message, t);
   }
});
